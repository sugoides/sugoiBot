# SugoiBot Documentation

Welcome to the documentation for **SugoiBot**, a modern, modular Discord bot framework built with Node.js.

## 🚀 Overview

SugoiBot is designed to be highly extensible through its modular architecture. It includes built-in systems for:

- **XP & Leveling**: Voice and text-based engagement tracking.
- **Economy**: A simple currency system with daily rewards and work commands.
- **Moderation**: Essential tools for server management (kick, ban, clear, warn).
- **Settings**: A dynamic configuration framework managed via SQLite.
